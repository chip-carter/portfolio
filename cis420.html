Development Project

CIS 420

This course focused on the detailed design and implementation phases of the system development life cycle. My team's project was to create a website that stores player statistics, medical data, strength training information, and more. In this course, I gained valuable experience in project management, web design/development, database design, agile development, and communication.

Key Topics Covered

  • Planning of information systems development activities
  • Monitoring and reporting progress with respect to plan expectations
  • Applying iterative development techniques to design and implement an information system
  • Implementation of an information system in accordance with design models and standard practices

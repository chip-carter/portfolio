Introduction to Network Security

CIS 480

This course deals with network security fundamentals, TCP/IP, virtualization, TCP/IP, network traffic signatures, routing fundamentals, cryptography, wireless networking fundamentals, wireless networking security, intrusion detection and prevention system, firewalls, firewall design and management, VPN concepts, and internet/world wide web security. I gained valuable experience in Kali Linux, DDoS attacks, packet analysis, packet tracing, and more.

Key Topics Covered

  • Threats to network security
  • Fundamentals of TCP/IP networking
  • IPv4 and IPv6 packet structure and packet fragmentation
  • Attack techniques against web sites and web users
  • Fundamental concepts and proccesses of risk analysis
